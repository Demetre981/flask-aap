# Flask Web App Tutorial

## Setup & Installtion

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install flask
```

## Go to app

```bash
cd app
```

```bash
$env:FLASK_APP = "__init__"
```

## Running The App

```bash
flask run
```

## Viewing The App

Go to `http://127.0.0.1:5000`
